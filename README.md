NHS Hospital Performance – Elective 18-Week Standard

This project compares elective waiting-time performance for Northumbria and Birmingham NHS Trusts using monthly data from NHS England. The aim was to see how each trust performs over time and how they stack up against the national average.

What’s included

Cleaned dataset for both trusts

Trend charts

Gap-vs-national chart

Yearly performance comparison

A simple HTML dashboard bringing everything together

Tools

Python (Pandas, Matplotlib) and basic HTML/CSS.
